Frontend Examples

Everything below is done with HTML, CSS, and pure Javascript! No libraries or frameworks.

Click the button below to change the background color to something random! The color will change back to white if you click it again. Try it several times to see several random combinations! This is done with Javascript event listeners, Math.random, a pure function, and a ternary statements.
Click the button below to change the text color. This is also done with Javascript event listeners. Click the button again to change back to the default color.
Click the button below to hide this whole box. An image appears in it's place, which can be clicked to undo the action. This is done with Javascript event listeners, removing and appending nodes. The image is only created and loaded after you remove the box.
Hover over this box to see the hidden detail content. This effect is done with CSS and HTML only, no Javascript.
